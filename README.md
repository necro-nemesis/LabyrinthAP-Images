# LabyrinthAP-Images
Snapshot images of LabyrinthAP.

Downgit online download manager will combine these github files into single zip file. Download into an empty folder and extract with 7z into multipart archive then extract the multipart archive with 7z again to generate the image file. Burn the image file with Etcher. See https://github.com/necro-nemesis/LabyrinthAP#quick-installer regarding instructions for logging into the GUI to use it.

Raspberry Pi 4 image Downgit link https://downgit.github.io/#/home?url=https://github.com/necro-nemesis/LabyrinthAP-Images/tree/main/Raspberry%20Pi%204
